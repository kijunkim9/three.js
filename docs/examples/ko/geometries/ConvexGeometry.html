[name]

[name] 를 사용하여 주어진 3D 벡터 배열에 볼록 껍질(Convex hull) 을 생성할 수 있습니다. 이 작업의 평균 시간 복잡도는 O(nlog(n)) 으로 간주됩니다.

코드 예제

const geometry = new ConvexGeometry( points ); const material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } ); const mesh = new THREE.Mesh( geometry, material ); scene.add( mesh );

예제

[example:webgl_geometry_convex geometry / convex ]

생성자

[name]( [param:Array points] )

점들 - [page:Vector3 Vector3s] 배열에 블록 껍질(Convex hull)이 나타내는 벡터들을 포함하고 있습니다.
